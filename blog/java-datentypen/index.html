Java Datentypen

Ein Überblick sowie Anwendungsbeispiele zu Java Datentypen.

13. 09. 2021
3 Minuten Lesezeit

Was sind Datentypen?

Datentypen sind Kategorien von Daten. Der Computer stellt Daten in Bits mit 0 und 1 dar. Um dabei Buchstaben, ganze Zahlen, Fließkommazahlen usw. Unterscheiden zu können und die Anzahl der Bit Länge zu begrenzen, ist eine Einordnung notwendig. In Java werden Datentypen in Primitive und Komplexe unterteilt. Diesen Datentypen werden dann Variabeln angehängt, welche ihren Inhalt definieren. Variabeln können dann im weiteren Programmverlauf ausgelesen und verändert werden.

Beispiel:

# int
15   = 00001111
# double
15,0 = 100000000101110000000000000000000000000000000000000000000000000

Was sind Variablen?

Variablen bezeichnen Werte, die sich während des Programmablaufs ändern. Operatoren sind Zeichen oder kleine Wörter, welche die Beziehung zwischen Variablen und Werten beschreiben. Den Namen der Variablen können wir selbst festlegen. Erlaubt ist fast alles, außer: Er darf kein Java-Wort sein, darf nur aus alphanumerischen Zeichen und dem Zeichen _ (Unterstrich) bestehen und nicht mit einer Ziffer beginnen.

Primitive Datentypen

Primitive Datentypen sind in Java unveränderlich eingebunden. Sie besitzen fest definierte Wertebereiche und sind gegen Objekte abgegrenzt. Die primitiven Datentypen wurden aus Performance-Gründen nicht als Klassen realisiert.

Ganze Zahlen

Name Wertebereich Bit Länge
byte -128 bis +127 8 bit
short -32768 bis +32767 16 bit
int -2147483648 bis +2147483647 32 bit
long -9223372036854775808 bis +9223372036854775807 64 bit

Flieskomma Zahlen

Name Wertebereich Bit Länge
float Einfache Genauigkeit (7 Nachkommastellen) 32 bit
double Doppelte Genauigkeit (16 Nachkommastellen) 64 bit

Zustände und Buchstaben

Name Wertebereich Bit Länge
boolean true oder false 8 bit
char ‘a’ oder ‘A’ (aA bis zZ) 16 bit

Komplexe Datentypen

Jeder komplexe Datentyp in Java kommt grundsätzlich aus einer Klasse. Dabei wird dieser im Regelfall durch einen Konstruktor erstellt. Java selbst liefert interne Datentypen wie String oder Array aus, welche wie die Primitive Datentypen über ein Gleichheitszeichen definiert werden können.

Name Zweck
String Sätze und Zeichenketten
Array Datenfeld - Container
enum Vordefinierte Konstanten als Variabel
BigDecimal Beliebig genaue Fließkommazahlen
BigInteger Beliebig große ganze Zahlen

Die wichtigsten im Überblick

Auch wenn jeder Datentyp seine Berechtigung hat, gibt es klare Favoriten. Die wichtigsten werde ich euch nun mit Funktion und Anwendung näher bringen. Ich habe spezifisch jene Datentypen ausgewählt, welche in ihrer werte Annahme am genausten sind und sich nicht doppeln.

Int

  • Primitiv
  • Für ganze Zahlen
  • -2147483648 bis +2147483647 (32 Bit)
  • Wird erstellt mit int Variabelname = ;
  • Standardwert 0
int x = 999999999;

Double

  • Primitiv
  • Für Flieskomma Zahlen
  • 16 Nachkommastellen (64 Bit)
  • Wird erstellt mit double Variabelname = ;
double y = 1.123123123;

Boolean

  • Primitiv
  • Für Zustände
  • true oder false (8 Bit)
  • Standardwert false
  • Wird erstellt mit boolean Variabelname = true/false;
boolean regen = true;
boolean sonne = false;

Char

  • Primitiv
  • Für Buchstaben aller Art (32 Bit)
  • Wird erstellt mit char Variabelname = ' ‘;
  • Buchstabe muss in ' ' geschrieben werden!
char peter = 'a';

String

  • Komplex
  • Für Zeichenketten
  • Wird erstellt mit String Variabelname = " “;
  • Alle Zeichen sind erlaubt
  • Buchstabe muss in " " geschrieben werden!
String beruf = "Millionär";

Array

  • Komplex
  • Darstellen von mehreren Daten in einer Liste
  • Wird erstellt mit Datentyp[] Variabelname ={ };
  • Die Angaberegeln der Datentypen sind dabei einzuhalten
  • Stellen im Array werden ab 0 gezählt
  • Auslesen mit Variabelname[Stelle im Array]
// Ganze Zahlen
int[] zahlen            = {1, 2, 3, 4, 5};
// Buchstaben
char[] buchstaben       = {'A', 'C', 'D'};
// Flieskomma Zahlen
double[] werte          = {1.12, 123.22, 0.33332};
// Zeichenkette
String[] zeichenfolgen  = {"Hallo, was geht?", "Java ist Cool", "Linux Rockt!"};

// Auslesen 
int zahl            = zahlen[3];
char buchstabe      = buchstaben[1];
double wert         = werte[0];
String zeichenfolge = zeichenfolgen[2];

Für alle Datentypen gilt

  • Enden mit Semikolon
  • Werden über das Gleichheitszeichen definiert
  • Typ steht am Anfang
  • Name ist frei wählbar
  • Werden zunächst initialisiert
  • Der Inhalt wird durch variabel bestimmt

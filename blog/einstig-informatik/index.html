Was ist Informatik?

Informatik ist eine Wissenschaft von der systematischen und oder automatischen Verarbeitung, Darstellung, Speicherung und Übertragung von Informationen mithilfe von Computern. Sie untersucht Verfahrensweisen und Methoden für die Verarbeitung von Informationen mithilfe von Algorithmen.

Teilgebiete der Informatik

Die Informatik untergliedert sich in 4 Teilbereiche, welche sich unterschiedlichen Themen und Tätigkeiten widmen.

Angewandte Informatik

  • Anwendung von technischer, praktischer und theoretischer Informatik in anderen Wissenschafts und Gesellschaftsbereichen.
  • Lösen alltäglicher informatischer Probleme
  • Zusammentreffen von Software und Hardware Entwicklung

Technische Informatik

  • Untersuchung des funktioneller Aufbaus der Hardware
  • logischer Entwurf von Rechentechnik
  • maschinennahe Programmierung

Praktische Informatik

  • Formulierung von Algorithmen als Programme
  • Betriebssysteme
  • Steuerungssystemen
  • Programmiersprachenentwicklung

Theoretische Informatik

  • Untersuchung und Darstellung der mathematischen Grundlagen der Informatik
  • Abstraktion und Modellbildung
  • Struktur, Verarbeitung, Übertragung und Wiedergabe von Informationen

Das EVA Prinzip

EVA ist die Abkürzung für: Eingabe Verarbeitung Ausgabe. Dies sind 3 Grundlegendsten interaktions- und Kommunikationsmöglichkeiten, welche zwischen Nutzer und Programm bestehen. Sie laufen wie folgt ab:

  1. Eingabe: Alle für das Programm relevanten Daten werden durch den Nutzer über eine GUI oder die Konsole Eingeben.

  2. Verarbeitung: Die vom Nutzer eingeben Daten werden durch das Programm erfasst und in einen Zielzusammenhang gebracht (berechnet).

  3. Ausgabe: Die berechneten Daten werden dem Nutzer über eine GUI oder auf der Konsole ausgeben.

Beispiel - Werte Berechnen in Java

Das unten gezeigte Programm kann nach dem EVA Prinzip die Summe, Differenz, Product und Quotient zweier werte Berechnen.

import java.awt.*;
import java.awt.event.*;
import javax.swing.*;

public class Rechner extends JPanel {
    private JLabel result;
    private JLabel label_w1;
    private JLabel label_w2;
    private JTextField wert1;
    private JTextField wert2;
    private JLabel label_options;
    private JComboBox options;
    private JButton calc;
    private String CalcOption;
    private int x;
    private int y;

    public Rechner() {
        // construct preComponents
        String[] optionsItems = { "Addieren", "Subtrahieren", "Multiplizieren", "Dividieren" };

        // construct components
        result = new JLabel("Ergebnis");
        label_w1 = new JLabel("Wert 1:");
        label_w2 = new JLabel("Wert 2:");
        wert1 = new JTextField(1);
        wert2 = new JTextField(1);
        label_options = new JLabel("Aktion:");
        options = new JComboBox<>(optionsItems);
        calc = new JButton("Berechnen");

        // Position
        result.setBounds(10, 10, 200, 20);
        label_w1.setBounds(10, 55, 100, 20);
        label_w2.setBounds(10, 105, 100, 20);
        wert1.setBounds(80, 50, 200, 40);
        wert2.setBounds(80, 100, 200, 40);
        label_options.setBounds(10, 155, 100, 20);
        options.setBounds(80, 150, 200, 40);
        calc.setBounds(10, 200, 270, 40);

        // Set Label for
        label_w1.setLabelFor(wert1);
        label_w2.setLabelFor(wert2);
        label_options.setLabelFor(options);

        // Actions
        options.addItemListener(new ItemListener() {
            @Override
            public void itemStateChanged(ItemEvent e) {
                if (e.getStateChange() == ItemEvent.SELECTED) {
                    CalcOption = e.getItem().toString();
                }
            }
        });

        calc.addActionListener(new ActionListener() {

            @Override
            public void actionPerformed(ActionEvent e) {
                getErgebnis();
            }
        });

        // adjust size and set layout
        setPreferredSize(new Dimension(290, 250));
        setBorder(BorderFactory.createEmptyBorder(10, 10, 10, 10));
        setLayout(null);

        // add components
        add(result);
        add(label_w1);
        add(label_w2);
        add(wert1);
        add(wert2);
        add(label_options);
        add(options);
        add(calc);
    }

    public void getErgebnis() {
        try {
            x = Integer.parseInt(wert1.getText());
            y = Integer.parseInt(wert2.getText());
        }

        catch (Exception e) {
            result.setText("Bitte nur Zahlen eingeben!");
            return;
        }

        if (CalcOption == null) {
            int Ergebnis = x + y;
            result.setText(Integer.toString(Ergebnis));
        }

        if (CalcOption == "Addieren") {
            int Ergebnis = x + y;
            result.setText(Integer.toString(Ergebnis));
        }

        if (CalcOption == "Subtrahieren") {
            int Ergebnis = x - y;
            result.setText(Integer.toString(Ergebnis));
        }

        if (CalcOption == "Multiplizieren") {
            int Ergebnis = x * y;
            result.setText(Integer.toString(Ergebnis));
        }

        if (CalcOption == "Dividieren") {
            int Ergebnis = x / y;
            result.setText(Integer.toString(Ergebnis));
        }

        return;
    }

    public static void main(String[] args) {
        try {
            UIManager.setLookAndFeel(UIManager.getSystemLookAndFeelClassName());
        } catch (ClassNotFoundException e) {
        } catch (InstantiationException e) {
        } catch (IllegalAccessException e) {
        } catch (UnsupportedLookAndFeelException e) {
        }
        JFrame frame = new JFrame("Rechner");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.getContentPane().add(new Rechner());
        frame.setResizable(false);
        frame.pack();
        frame.setVisible(true);
    }
}
